# Event Horizon Agentic Database Editor

A cross-platform application where users can upload Event Horizon game database, and use the convenient UI and AI Agent to make mods at unprecedented speeds.

## Usage
Clone the project, set up the environment variables in `.env` file.

If you have Golang installed, you can run the backend with `go run .` from the `backend/` directory, or `./run.sh` from the root directory.

If you don't have Golang installed, you can download and place the proper pre-built binary for your computer in the `bin/` directory and run it from that directory.
